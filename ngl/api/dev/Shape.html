Class: Shape

Shape

new Shape(name, params)

Class for building custom shapes.
Parameters:
Name Type Description
name String name
params Object parameter object
Properties
Name Type Description
aspectRatio Integer arrow aspect ratio, used for cylinder radius and cone length
sphereDetail Integer sphere quality (icosahedron subdivisions)
radialSegments Integer cylinder quality (number of segments)
disableImpostor Boolean disable use of raycasted impostors for rendering
openEnded Boolean capped or not
Example
var shape = new NGL.Shape( "shape", { disableImpostor: true } );
shape.addSphere( [ 0, 0, 9 ], [ 1, 0, 0 ], 1.5 );
shape.addEllipsoid( [ 6, 0, 0 ], [ 1, 0, 0 ], 1.5, [ 3, 0, 0 ], [ 0, 2, 0 ] );
shape.addCylinder( [ 0, 2, 7 ], [ 0, 0, 9 ], [ 1, 1, 0 ], 0.5 );
shape.addCone( [ 0, 2, 7 ], [ 0, 3, 3 ], [ 1, 1, 0 ], 1.5 );
shape.addArrow( [ 1, 2, 7 ], [ 30, 3, 3 ], [ 1, 0, 1 ], 1.0 );
var shapeComp = stage.addComponentFromObject( shape );
geoComp.addRepresentation( "buffer" );
