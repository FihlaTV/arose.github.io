Signals

visibilityChanged

Emitted when the component's *visibility* property has changed.

representationAdded

Emitted when a representation is added to the component.

representationRemoved

Emitted when a representation is removed from the component.

requestGuiVisibility

Used to communicate with GUI code while avoiding implementation details of the GUI to be present in the [page:Stage] layer.

statusChanged

Emitted when the component's *status* property has changed.

disposed

Emitted when a component is disposed of, that is at the end of the .dispose() method.

.updateRepresentations( what )

Triggers an update of data arrays (position, color, size) of all the components representations. For example, when the atom positions of a [page:Structure] have changed, the representations can be updated with component.updateRepresentations( { position: true } ). Mostly of internal use.
