[name]

A class for handling atom-based coloring.

Static signals

typesChanged

Emitted when the list of available color schemes changes.
// subscribe to changes of the list of available color schemes NGL.ColorFactory.typesChanged( function(){ console.log( "a color scheme has been added or removed" ); } )

Static methods

.addScheme( fn, [ label ] )

Returns an *id* that can be used as input for the *color* parameter of [page:StructureRepresentation structure representations] and to delete the scheme. The *fn*( atom ) function must return a hexadecimal RGB value. The *label* is for cosmetic use (i.e. in the GUI). To refer to the scheme internally always use the *id*.
// color atoms with a b-factor smaller 0.5 in green, else in red var myScheme = NGL.ColorFactory.addSelectionScheme( function( atom ){ if( atom.bfactor < 0.5 ){ return 0x00FF00; // green }else{ return 0xFF0000; // red } } ); // apply colorscheme to an existing representation repr.setParameters( { color: myScheme } ); // set colorscheme when creating a new representation component.addRepresentation( "cartoon", { color: myScheme } );

.addSelectionScheme( pairList, [ label ] )

Returns an *id* like .addScheme. Creates a coloring scheme based on *color*, *selection* pairs in *pairList*, where *selection* is a [page:Selection-language selection string] and *color* is either a color name, a hexadecimal RGB value or a CSS color string.
// color residues 1 to 100 in red, 101-200 in blue and the remaining in white var mySelectionScheme = NGL.ColorFactory.addSelectionScheme( [ [ "red": "1-100" ], [ "blue": "101-200" ], [ "white": "*" ] ] ); // apply colorscheme to an existing representation repr.setParameters( { color: mySelectionScheme } );

.removeScheme( id )

Deletes the scheme associated with *id*.

Constructor

[name]( type, [page:Structure structure] )

Available values for *type* are:
  • element (color by element)
  • resname (color by residue name)
  • ss (color by secondary structure)
  • atomindex (color by atom index)
  • residueindex (color by residue index)
  • chainindex (color by chain index)
  • modelindex (color by model index)
  • picking (color by picking id)
  • random (random color)
  • color (given uniform color, hexadecimal RGB value)
More schemes can be added with .*addScheme* and .*addSelectionScheme*.
// create a new color factory for coloring by the chainindex atom property var colorFactory = new NGL.ColorFactory( "chainindex", structure );

Methods

.atomColor( atom )

Returns a hexadecimal RGB color value.
// get color for an atom var hexColor = colorFactory.atomColor( atom );

.atomColorToArray( atom, array, offset )

Gets a color value like .*atomColor* buts write the three individual values for red, green and blue directly into an *array* at a specific *offset*
// fill an array with RGB values starting from position 3 var colorArray = new Array( 9 ); colorFactory.atomColorToArray( atom, colorArray, 3 ); // colorArray[ 3 ] = R // colorArray[ 4 ] = G // colorArray[ 5 ] = B
