There is an example configuration file for apache named *apache.config.sample* in the root directory of this package. To restart apache issue sudo /etc/init.d/apache2 restart

git

Follow these instructions to create a repository you can serve NGL from and push to it for updates.

Create directory mkdir ngl/ cd ngl/ Set-up repository git init git config receive.denyCurrentBranch ignore Create post-recieve file .git/hooks/post-recieve #!/bin/bash test "${PWD%/.git}" != "$PWD" && cd .. unset GIT_DIR GIT_WORK_TREE git reset --hard Make executable chmod +x .git/hooks/post-recieve Push from somewhere else (this is from another computer) git push ssh://user@server://path/to/repo/ngl master Checkout branch git checkout master
