Class OctahedronBuffer

Octahedron buffer. Draws octahedrons.

example

var octahedronBuffer = new OctahedronBuffer({ position: new Float32Array([ 0, 3, 0, -2, 0, 0 ]), color: new Float32Array([ 1, 0, 1, 0, 1, 0 ]), size: new Float32Array([ 2, 1.5 ]), heightAxis: new Float32Array([ 0, 1, 1, 0, 2, 0 ]), depthAxis: new Float32Array([ 1, 0, 1, 0, 0, 2 ]) })
